Aplikasi Untuk mengelola perpustakaan sekolah, dengan design yang masih sangat sederhana. Silakan dikembangkan. Untuk menggunakan : 1. Import Database lewat phpMyAdmin, 2. Ketikkan http://localhost/phpmyadmin/ 3. Buat database baru dengan nama : "db_perpus" 4. Lakukan import file "db_perpus.sql" di dalam folder "perpus" 5. Copykan seluruh folder "perpus" ke folder root webserver Anda (www atau htdocs) Terima kasih..
